1. 제작 방식 선택 가이드

A. 아트머그 기본 HTML

[장점]
  • 모바일 최적화 완벽 대응
  • 렉이나 로딩 지연 전혀 없음
  • 플랫폼 내 오류 발생 제로
[단점]
  • 디자인 일부 제한
  • 포트폴리오 등 내용 수정 시 직접 코드를 교체해야 함

B. iframe 임베드 방식

[장점]
  • 포트폴리오 실시간 업데이트
  • 재의뢰 시 구매자 수정 불필요
  • 제한 적은 디자인
[단점]
  • 모바일 일부 기능 오류 가능성 (스크롤 등)
  • 네트워크 상태에 따른 로딩 발생

2. 서비스 특장점 상세

A. 핵심 차별점

✅ PC · 모바일 완전 대응 UI 설계

화면 크기에 상관없이 정렬과 가독성 유지

✅ 플랫폼 제한 고려한 UX 최적화

아트머그 제약 내 최적의 인터페이스 구현

✅ 터치 인터페이스 기준 레이아웃

모바일에서도 클릭 편리한 버튼 배치

✅ 모바일 깨짐 방지 설계

고질적인 레이아웃 오류 원천 차단

B. 제작 능력 강조

✨ 레퍼런스 기반 1:1 커스텀 제작

참고하고 싶은 디자인 그대로 구현 가능

✨ 디자인 스타일 제한 없음

미니멀 / 카드형 / 브랜드형 모두 가능

✨ UI 설계 중심의 고도화 작업

단순 이미지가 아닌 탄탄한 구조적 설계

✨ 복잡한 인터랙션 구현 가능

고난도 UI 구성도 완벽하게 지원
